#3: IBM Year-Long Quantum Computing Course

By Tanishka Mehta

May 19, 2021

Through the online year-long course through IBM, taught by researchers from MIT, I was able to gain a deeper understanding of Quantum Computing; I learned about quantum circuits, Qiskit, Fourier Analyses, and various algorithms.

Quantum Circuits

Through the course, I learned how to compose circuits and apply gates to qubits. Through the IBM Quantum Circuit builder, I was able to view the qubit on the Bloch sphere and see the probability of each state occuring if the qubit were to collapse into a bit upon measurement. An example of creating a single-qubit circuit is displayed below, where a Hadamard gate (superposition gate) is applied. This results in a probability of each state occuring, 0 and 1, to be 50%.

Qiskit

As various quantum concepts were learned, such as how to apply quantum gates, I also learned how to code using the Qiskit library. I learned how to create circuits, apply gates to the qubits, and display the created circuit. An example is shown below.

# Create a Quantum Circuit acting on a quantum register of two qubits
circ_bell = QuantumCircuit(2)

# Add a H gate on qubit 0, putting this qubit in superposition.
circ_bell.h(0)

# Add a CX (CNOT) gate on control qubit 0 and target qubit 1, 
# putting the qubits in a Bell state.
circ_bell.cx(0, 1)

# Visualize the circuit
circ_bell.draw('mpl')

Fourier Analyses

Since any function can be represented or approximated with a linear combination of trignometric functions with varying frequences, a Fourier Transformation maps the time domain to its frequency domain. This helps determine the most important or the strongest frequencies in the signals.

In the image above, a hadamard gate is applied to a single qubit. This changes the basis of the qubit. For instance, if the qubit was in the state |0>, the Hadamard gate would create a superposition in the qubit, making it have the state |+>, thus changing the basis of the qubit. This is an example of a Single-Qubit Hadamard that applies Quantum Fourier Transform.

Algorithms

Due to the power of quantum computers and the advanced properties of qubits, many algorithms are being devised to optimize these advantages to make technology safer and more efficient.

Shor’s Algorithm

Shor’s algorithm, created by Peter Shor, is a polynomial-time quantum computer algorithm for integer factorization that can find the factors of a given integer. If a quantum computer with a sufficient number of qubits could operate without succumbing to quantum noise and other quantum-decoherence phenomena, then Shor’s algorithm could be used to break public-key cryptography schemes, such as RSA; RSA’s encryption and decryption keys are made by multiplying two prime numbers together. It’s security is based on the assumption that factoring large integers is computationally intractable. Thus far, this assumption is valid for classical computers. However, Shor’s algorithm shows that factoring integers is efficient on an ideal quantum computer, so it may be feasible to break RSA by constructing a large quantum computer. This is significant as it creates an incentive to develop safer cryptosystems.

Near-Term Algorithms

Since most algorithms assume perfect, error-less qubits and operations, these algorithms are beyond the capabilities of today’s devices. Today’s noise, imtermediate-scale quantum devices are limited by the number of qubits they can support (50-100 qubits). Therefore, Near-Term Algorithms is the idea that a quantum and classical computer can be combined to create a device that can find the optimal solution of any problem. For instance, it can find the minimal energy of a molecule, train machine learning models, create the optimal financial portfolio, and even factor large numbers. This is significant because as it can factor integers, it also has the capability to break RSA.
